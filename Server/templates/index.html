About this App

A2B

The Application "A2B" was created as part of a seminar at the FZI.
It was created to make use of the Yellow Cab Trip Data that is provided by New York City.
The Application tries to leverage realtime datastream as well as historic data in order to predict unexpected events. These events may be traffic, a construction site or some other incident. Beeing able to detect these events as well as providing information on several means of transportation, the user is always able to choose the best way to get from A2B.
